# 活动建议功能使用示例

## 功能介绍

新增的活动建议工具可以根据天气条件智能评估户外活动的适宜度，给出专业建议。

## 支持的活动类型

- **hiking** (爬山) - 默认活动
- **running** (跑步)
- **cycling** (骑行)
- **picnic** (野餐)

## 评分系统

系统综合考虑以下因素：
- 温度是否在理想范围内
- 风速是否适宜
- 未来3小时降水概率
- 当前天气状况（雨、雪、雷暴等）

评分等级：
- **EXCELLENT** (≥85分) - 天气绝佳，强烈推荐
- **GOOD** (≥65分) - 天气不错，适合活动
- **FAIR** (≥45分) - 天气一般，需要谨慎
- **POOR** (≥25分) - 不太适合，建议改期
- **AVOID** (<25分) - 天气恶劣，强烈不建议

## 使用示例

### 示例1：爬山建议

```json
{
  "city": "北京",
  "activity": "hiking"
}
```

返回：
```
📍 北京 - 爬山建议

🌡️ 当前天气：晴天
温度：22°C（体感 23°C）
湿度：45%
风速：12 km/h

✨ 适宜度评级：EXCELLENT

💡 建议：天气绝佳！非常适合爬山，记得带上足够的水和防晒用品。
```

### 示例2：雨天跑步建议

```json
{
  "city": "上海",
  "activity": "running"
}
```

返回：
```
📍 上海 - 跑步建议

🌡️ 当前天气：小雨
温度：18°C（体感 17°C）
湿度：85%
风速：8 km/h

✨ 适宜度评级：POOR

💡 建议：不太适合户外跑步，建议室内运动。

⚠️ 注意事项：
  - 天气状况不佳：小雨
  - 降水概率很高（75%）
```

### 示例3：骑行大风天

```json
{
  "city": "深圳",
  "activity": "cycling"
}
```

返回：
```
📍 深圳 - 骑行建议

🌡️ 当前天气：局部多云
温度：28°C（体感 30°C）
湿度：70%
风速：35 km/h

✨ 适宜度评级：FAIR

💡 建议：可以骑行但需谨慎，风力较大时减速慢行。

⚠️ 注意事项：
  - 风速偏大（35 km/h）
```

## 各活动的理想天气条件

### 爬山
- 温度：15-25°C
- 最大风速：30 km/h
- 最大降水概率：20%
- 避免天气：雨、雪、雷暴

### 跑步
- 温度：10-22°C
- 最大风速：25 km/h
- 最大降水概率：30%
- 避免天气：中大雨、雪、雷暴

### 骑行
- 温度：15-28°C
- 最大风速：20 km/h（骑行对风更敏感）
- 最大降水概率：15%
- 避免天气：雨、雪、雷暴

### 野餐
- 温度：18-26°C
- 最大风速：15 km/h
- 最大降水概率：10%
- 避免天气：雾、雨、雪、雷暴

## 如何在 Claude 中使用

配置好 MCP Server 后，可以直接问 Claude：

- "我想今天去北京爬山，看看天气适不适合"
- "上海今天适合跑步吗？"
- "我想在深圳骑行，天气怎么样？"
- "广州今天天气适合野餐吗？"

Claude 会自动调用 `get_activity_suggestion` 工具，给出专业的活动建议。
